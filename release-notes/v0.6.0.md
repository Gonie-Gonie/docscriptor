# v0.6.0

## Highlights

- Expanded editable report-panel layout controls on `BoxStyle(...)`:
  - explicit `width` and `unit`
  - per-side padding with `padding_top`, `padding_right`, `padding_bottom`, and `padding_left`
  - per-box `alignment`
  - `title_text_color`
- Made nested tables and figures render more consistently inside boxes across DOCX, PDF, and HTML.
- Added DOCX rendering for the shared fixed-sheet shape subset:
  - `Shape.rect(...)`
  - `Shape.ellipse(...)`
  - `Shape.line(...)`
- Updated the usage guide to prefer editable `Box(...)`, `Table(...)`, and `Figure(...)` report panels for tcolorbox-like layouts.

## Notes

- `Sheet(...)` remains available for fixed-layout form inserts, but report-style layouts should prefer editable flow blocks when Word editability matters.
